# Documentação para Desenvolvedores

<!-- Hi! Welcome to WordPress Playground Developer documentation. -->

Olá! Bem-vindo à documentação para desenvolvedores do WordPress Playground.

<p class="docs-hubs">A documentação do WordPress Playground está distribuída em quatro hubs (subsites) separados:</p>

-   [**Documentação**](/) – Introdução ao WP Playground, guias iniciais e seu ponto de entrada para a documentação do WP Playground.
-   [**Blueprints**](/blueprints) – Blueprints são arquivos JSON para configurar sua instância do WordPress Playground. Saiba mais sobre suas possibilidades no hub de documentação de Blueprints.
-   👉 [**Desenvolvedores**](/developers) (você está aqui) – O WordPress Playground foi criado como uma ferramenta programável. Descubra tudo o que você pode fazer com ele a partir do seu código neste hub de desenvolvedores.
-   [**Referência de API**](/api) – Todas as APIs expostas pelo WordPress Playground

## Navegando pelo hub de documentação para desenvolvedores

<!-- This docs hub is focused on Developers info and is divided into the following major sections: -->

Este hub de documentação é focado em informações para desenvolvedores e está dividido nas seguintes seções principais:

-   [Guia rápido para desenvolvedores](/developers/build-your-first-app): Comece sua jornada de desenvolvimento com o Playground explorando alguns exemplos do que você pode realizar usando as APIs do Playground.

-   [Desenvolvimento local](/developers/local-development): Descubra as ferramentas fornecidas pelo Playground para facilitar o processo de configuração e gerenciamento de sites WordPress e construir seus aplicativos.

-   [APIs do Playground](/developers/apis): Explore as principais APIs expostas pelo WordPress Playground para interagir com o Playground: [Query API](/developers/apis/query-api/), [Blueprints](/blueprints/), [Javascript API](/developers/apis/javascript-api/)

-   [Arquitetura](/developers/architecture): Os detalhes da arquitetura do WordPress Playground, incluindo seus vários componentes e ferramentas.

-   [Limitações](/developers/limitations/): Saiba mais sobre as limitações atuais do WordPress Playground, que incluem conexões de rede desabilitadas, dados temporários, peculiaridades de iframe, requisitos para execução de funções PHP e suporte parcial ao WP-CLI.
